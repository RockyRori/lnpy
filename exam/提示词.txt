简答题：解释主键和外键的区别。
答案: 
主键：主键是数据库表中每条记录的唯一标识符。它确保表中没有两行具有相同的主键值。主键不能接受空值，并能唯一标识表中的一条记录。
外键：外键是一个表中的属性，用于链接到另一个表的主键。外键可以接受多个空值，除非有明确的限制。

SQL题：使用SQL创建一个`department`表，包含以下字段：部门ID（整数主键）、部门名称（文本）、经理ID（整数外键）、起始日期（日期）。
答案: 
```sql
CREATE TABLE department (
    department_id INT PRIMARY KEY,
    department_name TEXT,
    manager_id INT REFERENCES employee(employee_id),
    start_date DATE
);
```
